## DATATYPES
In Python we use mainly three datatypes:
-> Integer(int) datatype stores whole numbers.
Ex: 1,2,3
-> Float-float datatype stores decimal values.
Ex: 22.22,4.32
-> String- (str) Anything Written in b/w quotes('',"",)known as string.
Ex: "aswini"
